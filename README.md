# NLWedding

A small static wedding website project built with HTML, CSS, and JavaScript.

## Project Structure

- `index.html` — main website page
- `css/style.css` — site styling and responsive layout
- `js/script.js` — interactive behavior and UI enhancements
- `images/` — project image assets

## Usage

1. Open `index.html` in your browser.
2. Edit the content in `index.html` to customize the wedding details.
3. Update styles in `css/style.css` and interactions in `js/script.js` as needed.

## Notes

- This is a static front-end site and does not require a server to run.
- For live preview while editing, use a simple local server or editor preview plugin.
